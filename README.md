# Projeto Criação de Sites

Este repositório contém projetos de **sites estáticos** desenvolvidos apenas com **HTML e CSS**, com foco em:
- Estrutura semântica e organizada
- Layouts responsivos
- Design limpo e moderno
- Boas práticas de estilização

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3

## 📂 Estrutura do projeto
- `/index.html` → Página principal
- `/css/` → Arquivos de estilo


## 🎯 Objetivo
Demonstrar como é possível criar sites profissionais utilizando apenas HTML e CSS, sem dependência de frameworks ou bibliotecas externas.
